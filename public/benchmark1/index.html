Introduction

This document describes a game called “Sadako”, which is designed to be a 2D side-scroller game. The game will employ sprite-based animation, tiled backgrounds, collision detection, physics, AI, side scrolling & gravity, and more basic 2D game techniques.

Technology

Sadako will be developed for the Browser using the Phaser javascript library, which is a open source framework for canvas and WebGL powered browser game. WebGL will be used as the underlying rendering technology. We’ll use FL Studio 20 to make music, Pro Create, Aseprite, and Piskel to make the sprites and other images. We also use Tiled to arrange the levels.

BackStory

Sadako is a little girl who is having bad dreams after her uncle played her the movie: The Ring.

She’s having nightmares! And, she’s living those nightmares.

Objective

Sadako’s goal is to find her Teddy bear in the nightmare. Sadako will face enemies such as demons and ghosts in her nightmare, and she will need to avoid these dangers.

She’ll have to navigate her way through the nightmare via jumps, double jumps, physics puzzles, and even some light combat.

Gameplay

The game will work like a side scroller, with Sadako able to run left and right, and jump up or double jump. Sadako also can push blocks, pull ropes, and even light fires to solve puzzles.

Terror level  --  0% - 100%

  • Once Sadako collides with an enemy, the terror level will increase by the value associated with the enemy.
  • If Sadako keeps touching the enemy, the terror level will increase in a constant speed (1 enemy unit of terror per 3 seconds). If Sadako is not being chased by any enemy, her terror level will slowly restore.
  • Sadako is immune to all terror damage for 3 seconds after taking damage.
  • On collide, the screen will briefly darken around the screen and shake (random chance for a scream)
  • If terror reaches 100%, Sadako becomes terrified and cannot continue.

Consumable Items:

  • Haunted doll -- Take the killing blow and teleport Sadako forward (Can go through walls)
  • Golden star sticker -- Makes Sadako invincible and can kill enemies on collision, limited duration
  • Milk Glass -- Instantly lowers terror by half
  • Cookies and Milk -- Instantly sets terror to 0%

Physics/Held Items:

  • Ropes -- Movable, Climbable
  • Blocks/Boxes/Planks/Balls/Triangles -- Movable, Walkable Platform, Moves other physics objects
  • Balloons -- Movable, Floating, Included Rope, Moves other physics objects
  • Uncle’s Lighter -- Holding the lighter will slow Sadako down, repel Ghosts / attract Moths

Checkpoint -- Keeps enemies away and acts as a new respawn point in the level.

Enemies/Obstacles:

  • Ghosts -- Cause a 10% increase in terror.
  • Spikes -- Cause a 25% increase in terror, moves Sadako to the spike’s return position.
  • Tables -- Hide from all enemies, enemy AI moves along, moving away from hiding place.
  • Moths -- Cause a 5% increase in terror per 3 seconds, but doesn’t activate damage immunity.
  • Evil Uncle -- Cause a 25% increase in terror when rushing starts
  • If he collides with Sadako, she gets a 50% terror damage, he stops for 3 seconds and she automatically moves a short distance towards the nearest table

Controls

A, S, D / <, v, >   --  Left, Down, Right

E / Shift / J         --  Grab

W / Space / ^     --  Jump

R / L                   --  Use Lighter (Toggle)

ESC / Pause      --  Pauses the game and displays the pause menu.

Graphical User Interface

Splash Screen

The splash screen GUI simply presents the game logo and “press any key to start”. Once pressed, a picture and some texts will fade in and tell the backstory of the game. There will also be a “Press ESC to skip”. After a few seconds, it will jump to the main menu.

In-Game Main Menu

        New game -- Start a new game at the level selected

        Help -- Display control keys and item introductions

About -- Show the information about the game developers

Help Screen

        Display control instructions and item descriptions

About Screen

        Display the backstory, important character and development members

In-Game Level-Completed Menu

        Next Level -- Start the next level

        Main Menu -- Go back to the main menu

In-Game Level-Failed Menu

        Retry Level -- Retry the level

        Main Menu -- Go back to the main menu

In-Game Pause Menu

        Resume -- Resume paused game

        Restart -- Restart the level from the beginning

        Help -- Display control keys and item introductions

        Main Menu -- Go to the main menu

In-Game GUI

Terror level bar -- Always above Sadako. Not shown if the terror level is 0. The bar increases from the center and goes toward both ends. Once the bar reaches both ends, it means the terror level is 100.

Pause button -- Always at the top-right of the screen. Once clicked, the game will pause and pop out a pause menu (does the same thing as the escape button on keyboard).

Tutorial -- Tutorial text will only be displayed in the tutorial level. It will show instructions and details about some game mechanisms.
